HTML Home

HTML or HyperText Markup language continues to be the standard markup language for structuring and presenting content on the World Wide Web. Over 30 years after its creation in 1991, this foundational language is essential to provide structure and meaning to the content of a webpage, allowing for the use of elements such as headings, paragraphs, links, images, and more. This website has been designed to act as a guide to provide information and resources in regards to what HTML is, its purpose, history, key information, and helpful resources for your further research.

Key Terms

Term Definition Example
HTML HyperText Markup Language, the standard markup language used in the creation of web pages. <!DOCTYPE HTML>
Tags A markup structure "<>" used to define HTML elements acts as a container. <head>,<body>,<p>,<h2>,<table>,<a>,<li>,<img>,<link>, etc.
Attributes A property of a tag that provides additional information about HTML elements, customizing their behvaior. name="value", style="color", width="100px", etc
Element Represents the different types of content within a webpage, beginning with a start tag, containing content such as the attributes and ending with an end tag. <h2> My header title style="color:blue" </h2> >
